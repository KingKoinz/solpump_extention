1. **Waits for game to start** ‚Üí Bot detects "betting open"
2. **Places bet** ‚Üí Enters your bet amount and clicks "Bet"
3. **Monitors multiplier** ‚Üí Watches the number go up (1.2x, 1.5x, 1.8x...)
4. **Cashes out at target** ‚Üí Clicks "Cashout" when it hits your target

## Troubleshooting

**Bot won't start:**
- Make sure you're on solpump.io
- Wallet must be connected
- Refresh the page and try again

**Bot not placing bets:**
- Check if betting is actually open
- Make sure you have enough SOL
- Try increasing the delay in settings

**Bot not cashing out:**
- Target might be too high (game crashes first)
- Try a lower target like 1.5x or 1.8x

**Stats not updating:**
- Refresh the extension popup
- Check browser console for errors
